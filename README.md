# DJ Transition Finder

A browser extension that reads your currently-playing track and helps you find
what to mix next — in key, in tempo.

## What it does

- Captures the track playing on SoundCloud (artist & title).
- Resolves the track's musical **key and BPM**.
- Scores harmonically compatible tracks on the Camelot wheel.
- Links out to real DJ set data so you can see what was played next.

## Data & attribution

Song key and BPM data is provided by [GetSongBPM](https://getsongbpm.com) — a free
database of song tempos and keys. This project uses the
[GetSongBPM API](https://getsongbpm.com/api) and credits
[getsongbpm.com](https://getsongbpm.com) as required by their terms of use.
